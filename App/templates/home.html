LMS PEERREVIEW CHECKER

A system is designed for analyzing irregular scores given by three reviewers. Each reviewer can give a score within the range of 0-10 points. The system has two functions.

The first function is to upload a CSV file that contains the following attributes: "Name, Assignment, Name of reviewer 1, score 1, Name of reviewer 2, score 2, Name of reviewer 3 and score 3." The data type for each attribute should only be "String" for name and "int" for score and assignment.

The second function of the system involves pulling data from Mango API. To access this function, you will need to provide a course ID, assignment ID, rubric ID, and professor's token.
